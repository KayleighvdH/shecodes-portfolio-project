World Clock

A responsive and user-friendly front-end application designed to help users effortlessly track the current time and date across various global time zones. Featuring a clean, modern interface, the project allows for easy selection of major cities worldwide and includes functionality to display the time for the user's current location, providing a seamless experience for managing international schedules.

Love Poem Generator

A creative and interactive front-end application that leverages artificial intelligence to generate unique, personalized love poems on demand. Users interact with a clean, responsive interface to submit prompts, themes, or keywords, and the AI model processes this input to instantly craft custom, emotionally resonant verses. This project showcases the powerful integration of generative AI within a modern web framework, delivering a delightful and inspiring user experience.

Weather Search

A professional and responsive front-end application designed to provide users with up-to-the-minute weather forecasts for any location worldwide. This project features an intuitive search interface and integrates a third-party weather API to display essential meteorological data, including temperature, humidity, wind speed, and daily conditions. It showcases robust API handling, asynchronous data fetching, and a clean, visually appealing design, ensuring a reliable and excellent user experience.
